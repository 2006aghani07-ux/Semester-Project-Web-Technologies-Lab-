# Gym Fitness Tracker
# 📌 Project Description

Gym Fitness Tracker is a web-based fitness management system developed using ASP.NET Core Web API, MongoDB, HTML, CSS, and JavaScript. The system helps gym administrators and members manage fitness-related information including:
1. User Management
2. Diet Plan Management
3. Progress Tracking
4. User Profiles
5. Authentication Pages (Login & Signup)
6. Admin Dashboard
The application provides a simple and interactive interface for tracking fitness goals, nutrition plans, and member progress.
